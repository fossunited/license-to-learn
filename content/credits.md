## Tech stack

- **[Zola](https://www.getzola.org)** :: static site generator (single Rust binary, no npm, no runtime JS framework)
- **Vanilla HTML/CSS/JS** :: No JS libs introduced early-on, claude wrote the simple JS files to serve its purpose for now. (As site grows more library will be used.)
  Agentic coding was used for this site to be built as required.
- **[qsv](https://github.com/jqnatividad/qsv)** :: validates every CSV against a sibling JSON Schema before it ships
- **GitHub Actions + GitHub Pages** :: build and deploy
